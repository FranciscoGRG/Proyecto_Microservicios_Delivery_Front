-Este es el Front del proyacto Delivery basado en microservicios

-Stack Tecnologico:
  -VSCode como IDE.
  -TypeScript como lenguaje y Angular como Framework
  -Framework de TailwindCss para los estilos.
  -Stripe como pasarela de pago.
  -SweetNotification para los alerts personalizados para los usuarios.
  -HTTPClient para las peticiones al Backend.

-Como ejecutarlo?
  -Primero ejecutar npm install
  -Una vez instalados los node modules, ejecutar ng serve -o.
